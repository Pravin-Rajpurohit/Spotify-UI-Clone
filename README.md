# 🎧 Spotify UI Clone

A simple Spotify-inspired UI built using **HTML, CSS, and JavaScript**. This project focuses on replicating the look and feel of Spotify’s interface while strengthening frontend fundamentals.

---

## 🚀 Features

* 🎵 Clean and responsive music player UI
* 📂 Sidebar navigation (Home, Search, Library)
* ▶️ Play/Pause button UI
* 📱 Basic responsive design

---

## 🛠️ Tech Stack

* **HTML** – Structure
* **CSS** – Styling & layout
* **JavaScript** – Basic interactivity

---

## 📸 Preview

A visually similar interface inspired by Spotify’s design (for learning purposes only).

---

## 📚 What I Learned

* Building structured layouts with HTML
* Styling complex UI using CSS (Flexbox/Grid)
* Handling basic DOM interactions with JavaScript
* Improving UI/UX thinking

---

## ⚠️ Disclaimer

This is a **frontend UI clone** created for educational purposes only. It does not include backend functionality or real music streaming.
